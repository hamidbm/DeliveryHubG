# Admin Module

The Admin module is used to configure global settings and reference data.

## Features
- Taxonomy management for document types and categories
- Bundle and capability configuration
- Wiki themes management
- AI provider settings and routing
- Global settings for environment and governance
- Admin-only access to operational controls

## Data
- Stored in collections such as `taxonomy_document_types`, `taxonomy_categories`, `bundles`, `wiki_themes`, and `ai_settings`
