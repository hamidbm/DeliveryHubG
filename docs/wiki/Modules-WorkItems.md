# Work Items Module

The Work Items module provides execution tracking similar to Jira.

## Features
- Work item hierarchy: Epic, Feature, Story, Task
- Kanban board with status columns
- Tree view for portfolio-level navigation
- List and roadmap views
- Milestone planning
- Activity history and assignments

## Statuses
- TODO
- IN_PROGRESS
- REVIEW
- DONE
- BLOCKED

## Data
- Stored in `work_items` collection
- Historical snapshots are tracked
