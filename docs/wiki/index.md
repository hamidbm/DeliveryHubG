# DeliveryHub Documentation

DeliveryHub is an internal software delivery and application portfolio management platform used to coordinate migration programs, delivery governance, architecture reviews, portfolio visibility, and executive reporting in one place.

This documentation is organized by audience.

## Start Here

- **Quick Setup**: for anyone who needs to install prerequisites and run DeliveryHub locally.
- **Product Guide**: for leaders, architects, PMO teams, delivery managers, and business users who want to understand what the product does and how to use it well.
- **Technical Documentation**: for engineers and developers working on the product.

## DeliveryHub At A Glance

DeliveryHub combines capabilities that are usually split across several tools:

- Jira-like work management and delivery tracking
- Confluence-like knowledge and documentation management
- LeanIX-like architecture and dependency visibility
- Planview-like application portfolio management
- executive dashboards, portfolio analytics, and AI-assisted decision support
