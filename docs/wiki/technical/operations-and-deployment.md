# Operations and Deployment

This page is the technical operations entry point.

## Runtime Basics

DeliveryHub supports local development with Node.js and Docker, and can also run through Docker Compose.

## Key Operational Areas

- bootstrap and seeding
- local and containerized startup
- scheduled jobs
- backup and restore
- admin audit surfaces

## Important Environment Variables

Common examples:

- `MONGO_URL`
- `JWT_SECRET`
- `AUTH_MODE`
- `AUTO_BOOTSTRAP_BASELINE`
- `INSTALL_SAMPLE_DATA`
- `AI_DEFAULT_PROVIDER`
